# Emisora Comunitaria — Sitio para GitHub Pages

Estructura mínima:

- index.html
- historia.html
- repertorio.html
- programacion.html
- locutores.html
- contacto.html
- styles.css
- script.js
- /assets (logo y fotos)

## Subir a GitHub y activar GitHub Pages (rápido)
1. Crea un nuevo repositorio en GitHub (por ejemplo `emisora-comunitaria`).
2. Clona el repo localmente o usa la interfaz web para subir archivos.
3. Sube todos los archivos y la carpeta `assets`.
4. En GitHub > Settings > Pages: selecciona la rama `main` (o `master`) y la carpeta `/ (root)` como fuente. Guarda.
5. GitHub generará una URL tipo `https://<tu-usuario>.github.io/<repo>/` (o `https://<tu-usuario>.github.io/` si eliges repo con mismo nombre que tu usuario en sitios de usuario).

## Personalizar
- Reemplaza `assets/logo.png` por tu logo.
- Reemplaza `assets/locutor1.jpg`, etc., por fotos reales.
- Edita textos y horarios directamente en los HTML.
- Para formularios avanzados, conecta un servicio de backend o formularios (Formspree, Netlify Forms, etc.).

## Notas
- Si quieres que al entrar muestre un reproductor en vivo (stream), te puedo mostrar cómo integrar el <audio> o reproductor embebido (necesitarás la URL del stream).
- Para mejoras avanzadas (CMS, blog, multi-idioma), puedo ayudarte a integrar Netlify, Forestry o similar.
